THIS FOLDER CAN BE SEEN AS A STANDARD ANSWER FOR WISE2013_TRACK1
There are six documents in this folder:
1、document_1500:1500 files that are really used for checking.
2、standard_result.txt:This txt file contains all the tag information for the 1500 files,like(docnum, mentionoff, entityname, 
mention, type in which 2 stands for PN).
3、redirect.txt:As we all know, there are many "redirect" pages in wikipedia, different forms of phrases may point to the same
mention, so in this TXT, we can see redirect relations appearing in 1500 TXT;
4、answer.txt:You can put your tagging information in this TXT so that the program "Testaccuracy.java" can work it out.
5、Testaccuracy.javaThis java program is what we use to test your result. It needs three input files, named: answer.txt,redirect.txt, 
standard_result.txt.
